# AgenticAI

A learning repository for exploring **agentic AI** — AI systems that can plan, use tools, and act autonomously to accomplish goals.

## What is agentic AI?

Traditional AI models respond to a single prompt with a single answer. **Agentic AI** goes further: an agent can break a task into steps, decide which tools to use (search, code execution, APIs, databases, and more), observe results, and iterate until the job is done. This pattern powers assistants, coding agents, research bots, and other systems that operate with a degree of autonomy rather than one-shot generation.

## About this repo

This repository holds coursework and experiments from a **two-week certificate in Agentic AI**. It is a place to practice building and studying AI agents — from basic tool use and orchestration to more advanced multi-step workflows.

Projects here are primarily **Python**-based. As work progresses, this repo will grow with notebooks, scripts, and agent implementations developed during the certificate.
