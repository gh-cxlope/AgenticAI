What happens when you ask a question

This app uses a multi-agent system built on top of California native plant knowledge.

  1. You ask a question in the chat or click a pollinator fun-fact card.
  2. An orchestrator reads your question and chooses the best expert agent.
  3. That agent uses either RAG over CSV data or web search, then GPT writes the answer.
  4. The response appears in the chat, labeled with the route that was used.

Slide 2 of 3

The four expert agents

Plant retriever

Landscaping & plant traits

Prompt focus: answer only from retrieved plant records; include scientific and common names, height, and spread; stay practical and concise.

Pollinator expert

Wildlife & pollinator support

Prompt focus: answer only from pollinator CSV data; mention bees, butterflies, hummingbirds, or monarchs; recommend plants by pollinator value.

Fun fact agent

One-sentence trivia

Prompt focus: use DuckDuckGo web search, then return exactly one playful, surprising sentence about plants or pollinators.

Both + synthesizer

Combined answer

When a question needs plant details and pollinator info, both RAG agents run and a synthesizer merges their answers into one response.
